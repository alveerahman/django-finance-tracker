# 💰 Django Personal Finance Tracker

A robust web application to track daily income and expenses, built using the **Django** framework. This project demonstrates backend skills like **ORM Aggregation**, **MVT Architecture**, and **Database Management**.

## ✨ Features
- **Dashboard Summary:** View real-time balance, total income, and total expenses.
- **Transaction History:** List of all financial activities ordered by date.
- **Smart Aggregation:** Calculations are performed at the database level using Django's `Sum` function for high performance.

## 🛠️ Tech Stack
- **Backend:** Django 4.x (Python)
- **Database:** SQLite
- **Frontend:** Basic HTML/CSS

## 🚀 How to Run locally
1. Clone the repo: `git clone https://github.com/alveerahman/django-finance-tracker.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start server: `python manage.py runserver`
